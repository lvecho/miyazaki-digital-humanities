📚 Project Information

Digital Humanities Final Project

Miyazaki Hayao: Animation & Digital Analysis

🎯 Project Objectives

This digital humanities project explores the intersection of traditional Japanese animation and modern computational analysis techniques. Our primary objectives include:

  • Analyzing narrative structures in Miyazaki's films using NLP
  • Visualizing character relationships and thematic elements
  • Creating an interactive digital archive of Studio Ghibli works
  • Demonstrating the value of computational methods in film studies

🔬 Research Methodology

Our interdisciplinary approach combines traditional film analysis with cutting-edge digital tools:

  • Text Mining: Screenplay analysis using Python NLP libraries
  • Data Visualization: Interactive charts and network graphs
  • Sentiment Analysis: Emotional arc mapping throughout films
  • Web Development: Creating accessible digital presentations

📖 Academic Context

This project contributes to the growing field of digital humanities by:

  • Bridging Eastern and Western analytical traditions
  • Applying computational methods to visual storytelling
  • Creating replicable methodologies for film analysis
  • Demonstrating the cultural significance of animation as literature

Course: Digital Humanities Methods and Applications
Institution: University Digital Studies Program
Semester: 2025

🏆 Expected Outcomes

Upon completion, this project will deliver:

  • A comprehensive digital archive of Miyazaki's major works
  • Quantitative analysis of narrative and thematic patterns
  • Interactive visualizations accessible to general audiences
  • A replicable framework for analyzing animated films
  • Scholarly contributions to animation and digital humanities studies
